🔥 Firebase 연결 설정

앱·컴퓨터 실시간 동기화를 위해 Firebase 정보를 입력해주세요.
한 번만 입력하면 이 기기에 저장됩니다.

① Firebase 프로젝트 만들기
console.firebase.google.com 접속 → 새 프로젝트 생성
② Realtime Database 활성화
좌측 메뉴 → Build → Realtime Database → 데이터베이스 만들기
규칙(Rules) 탭 → 아래 내용으로 교체 후 게시:
{ "rules": { ".read": true, ".write": true } }
③ 프로젝트 설정 → 내 앱 → 웹 앱 추가
생성된 databaseURL 과 apiKey 를 아래에 붙여넣기
